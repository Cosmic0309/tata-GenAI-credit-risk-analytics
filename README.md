# AI-Powered Credit Risk Analytics

An end-to-end credit risk analytics case study developed as part of the Tata GenAI Powered Data Analytics Job Simulation by Forage.

This project demonstrates how data analytics, predictive modelling, business intelligence, and Responsible AI can be applied to improve customer delinquency prediction and collections strategy.

---

## Project Overview

Financial institutions often rely on reactive collections processes after customers have already missed payments.

This project explores how predictive analytics can identify high-risk customers earlier, enabling proactive intervention while maintaining fairness and transparency.

The project follows an end-to-end analytics workflow from raw data exploration to business recommendations.
